# MVP: Сервис бронирования услуг (Booking Service)

## Описание
Это минимально жизнеспособный продукт (MVP) для записи клиентов в салон красоты/барбершоп.
Приложение позволяет администратору добавлять услуги и просматривать брони, а клиентам - выбирать свободное время и записываться.

## Технологии
- HTML5
- CSS3
- JavaScript (ES6)

## Запуск приложения
1. Скачайте или клонируйте репозиторий.
2. Откройте файл `index.html` в любом современном браузере (Chrome, Firefox, Edge, Safari).

*Примечание: Все данные хранятся временно в памяти браузера. При обновлении страницы добавленные услуги и бронирования сбросятся до демо-данных.*

## Использование
1. **Администратор:** В блоке "Панель администратора" добавьте услугу (например, "Стрижка", 40, 1200). Ниже вы увидите список броней.
2. **Клиент:** В блоке "Запись онлайн" выберите услугу, дату, нажмите "Показать свободное время".
3. Нажмите на свободный слот, введите имя и телефон и подтвердите запись.
4. Новая запись сразу появится в списке броней у администратора.
